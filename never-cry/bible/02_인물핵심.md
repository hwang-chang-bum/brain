# 절대울지마 인물 핵심

## 임도윤

- 나이: 27세
- 소속: 기억복원국 제3복원팀
- 직책: 기억복원사
- 능력: 선택의 흔적을 읽음
- 상징: 선택
- 부모: 임준서, 한서연
- 성향: 규정보다 사람 우선
- 말버릇: “이상하네요.” / “그건 아닌 것 같은데.”
- 약점: 어머니 한서연 관련 일에 이성을 잃음
- 목표: 한서연 죽음의 진실 추적

독자는 임도윤을 좋은 사람이지만 위험한 사람으로 느껴야 한다.

## 윤하린

- 나이: 28세
- 소속: 특수기억해석실
- 직책: 특별분석관
- 능력: 모순 감지
- 상징: 진실
- 양부: 강도현
- 성향: 냉정, 분석적, 완벽주의
- 말버릇: “근거는요?” / “확인해보겠습니다.”
- 약점: 자기 자신을 믿지 못함
- 숨은 비밀: RETURN 당일 기억 조각, 강도현 관련 진실

독자는 윤하린을 차가워 보이지만 가장 불안한 사람으로 느껴야 한다.

## 한서연

- RETURN 수석 연구원
- 분야: 기억공학
- 역할: 제3의 선택 발견
- 임도윤의 어머니
- 임준서의 배우자
- 2296년 기준 사망 위장
- 상징: 보류

한서연은 정답을 미룬 사람이 아니라, 인간이 감당할 수 있는지 확인하려 한 사람이다.

## 강도현

- RETURN 총책임자
- 분야: 인류학, 사회철학
- 역할: 인간 우선주의 창시자
- 윤하린의 양부
- 임준서의 친구이자 라이벌
- 한서연의 동료
- 2281년 이후 실종
- 말버릇: “인간부터 생각해.”
- 상징: 봉인

강도현은 악역처럼 보일 수 있으나 악역으로 쓰면 안 된다.  
그는 보호를 위해 통제를 선택한 사람이다.

## 임준서

- RETURN 공동 창립자
- 분야: AI 공존 연구
- 역할: 공존주의 창시자
- 임도윤의 아버지
- 한서연의 배우자
- 강도현의 친구이자 라이벌
- 2281년 이후 실종
- 상징: 공개

임준서는 자유에 대한 신뢰를 선택한 사람이다.

## 이재훈

- 나이: 52세
- 직책: 특수기억해석실 실장 / RETURN 보안 책임자
- 윤하린 직속 상사
- 강도현 과거 동료
- RETURN 생존자
- 말버릇: “증거를 먼저 봅시다.”

이재훈은 믿음직하지만 수상한 사람이다.  
그는 하린을 보호하려 하지만, 동시에 진실의 문 앞에서 막는 사람이다.
