# 원고 보관 / 파일명 규칙

## 목적

절대울지마 원고와 설정 문서를 GitHub에서 안정적으로 관리하기 위한 파일명 규칙이다.

초안, 수정본, 폐기본, 확정본을 섞지 않는다.  
원고와 설정 문서도 섞지 않는다.

---

## 기본 폴더 구조

```text
never-cry/
├─ bible/          # 세계관, 연표, 인물, RETURN 핵심 설정
├─ rules/          # 집필 규칙, 작풍 금지, 충돌 검증
├─ style/          # 문체 샘플, 금지문장 변환표
├─ episodes/       # 확정본 또는 회차별 설계
├─ drafts/         # 초안/수정본/폐기본
├─ mystery/        # 떡밥, 공개 단계표
├─ relationships/  # 관계 진행표
├─ characters/     # 등장인물 사용 규칙
├─ history/        # 폐기/수정 이력
└─ prompts/        # 새 채팅 리프레시 프롬프트
```

## 원고 저장 원칙

### 1. 확정본

확정본은 `episodes/final/`에 저장한다.

```text
never-cry/episodes/final/018_D17.md
never-cry/episodes/final/019_분석자태그.md
never-cry/episodes/final/020_응답값.md
```

확정본에는 현재 기준으로 살릴 원고만 둔다.

---

### 2. 초안

초안은 `drafts/raw/`에 저장한다.

```text
never-cry/drafts/raw/018_D17_v01.md
never-cry/drafts/raw/018_D17_v02.md
```

초안은 완성도가 낮아도 보존 가능하다.  
단, 초안임을 명확히 표시한다.

---

### 3. 수정본

사용자 피드백 후 고친 원고는 `drafts/revised/`에 저장한다.

```text
never-cry/drafts/revised/018_D17_rev01.md
never-cry/drafts/revised/020_응답값_rev02.md
```

수정본 상단에는 반드시 수정 이유를 적는다.

```md
> 수정 이유:
> - 설명형 문장 과다
> - 사건이 하린에게 돌아오지 않음
> - D-17 단서를 현재 사건에서 재구성
```

---

### 4. 폐기본

폐기본은 `drafts/discarded/`에 저장한다.

```text
never-cry/drafts/discarded/018_통화사건_폐기.md
never-cry/drafts/discarded/020_응답값_웹툰형_폐기.md
```

폐기본은 버리지 않는다.  
왜 실패했는지 학습 자료로 남긴다.

폐기본 상단에는 반드시 적는다.

```md
> 폐기 이유:
> - 윤하린 에피소드로 기능하지 않음
> - 단서가 하린에게 돌아오지 않음
> - 웹툰 설명형 문장 반복
```

## 파일명 규칙

### 기본 형식

```text
회차_짧은제목.md
```

예시:

```text
018_D17.md
019_분석자태그.md
020_응답값.md
021_말하지않는사람들.md
```

---

## 버전 규칙

### 초안

```text
018_D17_v01.md
018_D17_v02.md
```

### 수정본

```text
018_D17_rev01.md
018_D17_rev02.md
```

### 폐기본

```text
018_D17_폐기_통화사건.md
020_응답값_폐기_웹툰형.md
```

### 확정본

```text
018_D17.md
020_응답값.md
```

확정본에는 버전 번호를 붙이지 않는다.

## 회차 문서 상단 메타 정보

각 회차 원고 상단에는 아래 메타 정보를 붙인다.

```md
# 22화. 같은 사건, 다른 눈

- 상태: 초안 / 수정본 / 확정본 / 폐기본
- 기준 문서:
  - rules/08_22_30화_집필규칙.md
  - style/03_절대울지마_문장체크리스트.md
- 회차 기능:
  - 도윤과 하린이 같은 D-17 사건을 다르게 본다.
- 공개 가능 단서:
  - D-17
  - 보호자 유도 발화
  - 선택의 흔적
- 공개 금지:
  - G.D.H. 정체 확정
  - 강도현 이름 직접 연결
  - RETURN 전체 설명
```

## 커밋 메시지 규칙

GitHub 커밋 메시지는 짧고 명확하게 쓴다.

### 설정 문서 추가

```text
Add Never Cry D-17 clue disclosure rules
```

### 원고 초안 추가

```text
Add draft for episode 22
```

### 원고 수정

```text
Revise episode 20 to reduce webtoon-style exposition
```

### 폐기본 보관

```text
Archive discarded episode 18 phone-call draft
```

---

## 금지

- 확정본 폴더에 폐기본 넣기
- 같은 파일명을 덮어쓰기
- `최종`, `진짜최종`, `최최종` 사용
- 설정 문서를 episodes에 넣기
- 원고를 bible에 넣기
- 폐기 이유 없이 폐기본 저장
