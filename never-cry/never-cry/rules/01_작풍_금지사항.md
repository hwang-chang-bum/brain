# 01. 작풍 금지사항

이 문서는 **절대울지마** 집필 중 반복적으로 발생한 실패 패턴을 막기 위한 금지 규칙이다.

## 1. 웹툰 설명형 금지

다음 문장 패턴을 반복하지 않는다.

```text
하린은 보았다.
송유진은 말했다.
박현우가 물었다.
아무도 대답하지 않았다.
화면이 바뀌었다.
하린은 멈췄다.
```

이런 문장은 장면을 보여주는 것이 아니라 웹툰 콘티의 컷 설명처럼 보인다.

### 금지 이유

- 인물이 살아 움직이지 않고 장면 설명으로만 소비된다.
- 감정이 문장 안에서 발생하지 않는다.
- 독자가 소설을 읽는 느낌보다 장면 요약을 받는 느낌을 받는다.

### 대체 방향

행동을 나열하지 말고, 장면의 압력을 문장에 녹인다.

```text
X: 하린은 화면을 보았다.
O: 화면 위 문장이 하린의 호흡을 한 박자 늦췄다.

X: 아무도 대답하지 않았다.
O: 대답이 없는 게 아니라, 모두 같은 이름을 피하고 있었다.

X: 송유진이 말했다.
O: 송유진의 목소리가 먼저 하린의 손목을 붙잡았다.
```

## 2. 자료 분석형 전개 금지

다음 구조를 반복하지 않는다.

```text
자료를 연다
→ 기록을 확인한다
→ 코드/로그/숫자를 분석한다
→ 누군가 들어와 의미를 설명한다
→ 마지막에 떡밥 문장 하나를 던진다
```

### 금지 이유

이 구조는 소설이 아니라 사건 처리 보고서처럼 보인다.  
윤하린 중심 구간에서 필요한 것은 자료 해설이 아니라 **사건이 하린을 향해 돌아오는 압박**이다.

## 3. 회의록식 대사 금지

다음 식의 대사 흐름을 피한다.

```text
"이건 D-17입니다."
"그 뜻은 무엇입니까?"
"보호자 유도 발화입니다."
"현행 분류표에는 없습니다."
"그러면 과거 태그군요."
```

### 금지 이유

인물이 정보를 주고받기 위한 장치로만 보인다.  
대사가 갈등, 회피, 압박, 숨김을 만들어야 한다.

### 대체 방향

정보를 말로 설명하기보다 인물의 반응으로 먼저 드러낸다.

```text
"D-17입니다."

송유진의 손이 먼저 멈췄다.
하린은 그 짧은 정지를 놓치지 않았다.

"처음 듣는 코드라고 하셨죠."
"그래."
"방금은 처음 듣는 사람처럼 보이지 않았습니다."
```

## 4. 떡밥 컷 엔딩 금지

화 마지막에 다음 식으로 끝내는 것을 남발하지 않는다.

```text
그 화면에는 세 글자가 떠 있었다.
G.D.H.
```

### 금지 이유

웹툰식 마지막 컷처럼 보이고, 장면 전체가 떡밥을 위해 존재하게 된다.

### 대체 방향

떡밥보다 인물의 상태 변화로 끝낸다.

```text
세 글자는 사라졌지만, 하린은 누군가가 문장을 덮어 버렸다는 느낌을 지울 수 없었다.
그것도, 자신보다 먼저.
```

## 5. 핵심 금지 단어 남발

아래 단어들은 필요할 때만 제한적으로 사용한다.

- 감정 키
- 판정
- 보류
- 근거
- 기록
- 통지 문구
- 시스템
- 재분류
- 보호자 동의
- 로그
- 코드
- 태그

### 사용 원칙

이 단어들이 한 장면에 몰리면 보고서가 된다.  
웹소설 장면에서는 사건, 호흡, 침묵, 말하지 않는 반응이 먼저 와야 한다.

## 6. 인물 감정 직접 해설 금지

다음 식의 해설을 피한다.

```text
하린은 불안했다.
하린은 혼란스러웠다.
하린은 강도현을 떠올렸다.
```

### 대체 방향

감정은 신체, 시선, 선택 지연, 말의 속도로 보여준다.

```text
하린은 바로 대답하지 않았다.
평소라면 가장 먼저 나왔을 말이 목 안쪽에서 한 번 걸렸다.
```

## 7. 윤하린 구간 특수 금지

13~20화 윤하린 중심 구간에서는 다음을 금지한다.

- 방주 정체 공개
- 노아 정체 공개
- 외계문명 정체 공개
- RETURN 최종 철학 공개
- 강도현 이름 직접 확정 남발
- 2281년 사건 강의식 설명
- 이재훈이 친절하게 과거를 설명하는 장면

## 8. 판단 기준

초안을 쓴 뒤 아래 질문에 하나라도 걸리면 다시 쓴다.

1. 이 장면이 자료 분석으로만 진행되는가?
2. 인물들이 정보를 설명하기 위해서만 말하는가?
3. 하린의 과거 단서가 현재 사건에서 자연스럽게 나온 것이 아니라 억지로 붙은 것인가?
4. 문장 대부분이 “누가 무엇을 했다” 형태인가?
5. 마지막이 떡밥 컷처럼 끝나는가?
6. 이 장면을 지워도 인물 관계나 사건 압박이 크게 변하지 않는가?

하나라도 해당하면 장면 기능이 약한 것이다.
