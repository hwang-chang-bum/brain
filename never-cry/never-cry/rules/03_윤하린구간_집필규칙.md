# 03. 윤하린 구간 집필규칙

이 문서는 13~20화 및 그 이후 이어지는 윤하린 중심 구간의 진행 원칙을 정리한다.

## 1. 구간 목표

13~20화는 윤하린의 비밀을 공개하는 구간이 아니다.

목표는 다음 하나다.

> 윤하린이 사건을 분석하는 사람이 아니라, 이미 사건 안에 들어와 있는 사람이라는 사실을 독자가 눈치채게 한다.

따라서 직접 설명하지 않는다.  
확정하지 않는다.  
과거를 강의하지 않는다.

## 2. 현재 고정 시점

- 현재 연도: 2296년
- RETURN 사건: 2281년
- 윤하린: 28세
- 임도윤: 27세
- 윤하린은 강도현의 양녀
- 윤하린은 특수기억해석실 특별분석관
- 임도윤은 기억복원국 제3복원팀 기억복원사
- 도윤과 하린은 아직 불편한 협력자 단계

## 3. 윤하린 핵심 상태

윤하린은 다음 감정을 동시에 가진다.

- 강도현에 대한 존경
- 강도현에 대한 원망
- 강도현에 대한 그리움
- 자기 기억에 대한 불신
- 자신의 판단에 대한 과도한 통제
- 감정을 드러내지 않으려는 습관

하지만 이 감정들을 직접 설명하지 않는다.

## 4. 사용 가능한 하린 키워드

13~20화 및 이후 구간에서 사용할 수 있는 키워드:

- 괜찮지 않으면
- 괜찮다고 말하지 않아도 된다
- D-17
- 보호자 유도 발화
- 분리 보류
- 분석자 일치
- 본인 반응값 대조
- 지정 배정
- G.D.H.
- 비 오는 날
- 두 시간 늦었지만 결국 온 사람
- 미안하다
- 언젠가 알게 될 거다

단, 이 키워드를 한꺼번에 쓰지 않는다.  
한 화에 중심 키워드는 하나, 보조 키워드는 최대 하나만 쓴다.

## 5. 사용 금지 또는 제한 키워드

아래는 직접 공개 금지 또는 극히 제한한다.

- 방주 정체
- 노아 정체
- 외계문명 정체
- RETURN 최종 철학
- 강도현의 최종 선택 확정
- 한서연의 제3의 선택 상세 설명
- 2281년 사건의 전모 강의
- 윤하린 기억 분리의 확정 설명

## 6. 사건 작동 방식

윤하린 구간의 사건은 반드시 아래 구조를 가진다.

```text
현재 사건 발생
→ 평범한 업무처럼 시작
→ 사건 안에서 이상한 단서 발견
→ 단서는 일반 분석관에게는 사소해 보임
→ 하린에게만 과도하게 걸림
→ 팀원 중 누군가 그 반응을 눈치챔
→ 하린은 단서에서 빠지려 하지만 사건이 하린을 다시 부름
```

## 7. 사건은 하린의 단서로 돌아와야 한다

잘못된 구조:

```text
현재 사건 발생
→ 이상한 통화
→ 하린이 보류 판정
→ 끝
```

이 구조는 윤하린 에피소드로 기능하지 않는다.

올바른 구조:

```text
현재 사건 발생
→ 이상한 통화
→ D-17 태그 발견
→ D-17이 구형 사전에서 보호자 유도 발화로 확인
→ 현행 분류표에는 없음
→ 사건이 하린에게 지정 배정
→ 지정 사유가 분석자 일치로 표시
```

즉, 사건은 반드시 **하린에게 돌아오는 단서**를 남겨야 한다.

## 8. 팀원 역할

### 송유진

- 하린의 제동장치
- 절차와 보호를 동시에 담당
- 무언가 알고 있지만 다 말하지 않음
- 하린이 위험한 곳으로 들어가면 먼저 막음

### 박현우

- 정보 전달과 분위기 담당
- 가벼워 보이지만 중요한 내부 사전, 잡음, 이상 로그를 찾아냄
- 독자에게 숨 쉴 틈을 주되, 장면을 희화화하지 않음

### 이나경

- 논리 대립축
- 통계와 시스템 관점에서 이상함을 짚음
- 감정적 배려는 서툴지만 정확한 말을 함
- 하린의 반응이 사건의 일부가 되었음을 가장 먼저 논리화함

### 이재훈

- 침묵하는 관문
- 하린의 과거와 강도현 관련 진실을 일부 알고 있음
- 직접 설명하지 않음
- 말하지 않는 태도로 단서가 됨

### 임도윤

- 선택의 흔적을 읽는 사람
- 하린과 상반된 방식
- 21화 이후 D-17 사건에 합류
- 하린은 도윤을 불편해하지만, 도윤의 방식이 필요하다는 사실을 부정하기 어렵다

## 9. 18~21화 진행 기준

### 18화: D-17

- 평범한 감정 키 재분류 요청으로 시작
- 대상자가 “괜찮아요” 반복
- 세 번째 발화 직전 “말해” 음성 확인
- 자동 보조 태그 D-17 발견
- 구형 내부 사전에서 D-17 = 보호자 유도 발화 확인
- 사용 중지, 2281년 이후 암시
- 하린은 D-17을 보고서에서 지우지 않음

### 19화: 분석자 태그

- 송유진이 하린을 막으려 함
- D-17 사건이 하린에게 지정 배정
- 지정 사유: 분석자 일치
- 상세 로그: 본인 반응값 대조
- 사건이 대상자가 아니라 하린의 반응을 기준으로 움직였다는 의심
- 하린은 공식 기록을 남김

### 20화: 응답값

- 하린의 단말에 분석자 반응값 재측정 알림
- 입력창 없이 문장 제시: “괜찮지 않으면”
- 하린이 후속 문장을 떠올리려는 순간 세션 차단
- 외부 개입자 코드: G.D.H.
- 이름 확정은 하지 않음
- 누군가 하린보다 먼저 문장의 뒤를 막았다는 느낌으로 종료

### 21화: 말하지 않는 사람들

- 하린과 송유진이 이재훈 실장실로 감
- 이재훈은 이미 세션 강제 종료를 알고 있음
- 이재훈은 “그 이름은 아직 부르지 마십시오”라고 말함
- 하린은 이재훈이 이니셜을 이름으로 알고 있음을 눈치챔
- 오래된 봉투에서 D-17 / 보호자 유도 발화 / 분리 보류 일부 노출
- 이재훈은 끝까지 설명하지 않음
- 임도윤 합류
- 이후 하린의 모순 감지와 도윤의 선택 흔적 읽기가 함께 필요해짐

## 10. 화별 종료 원칙

각 화는 아래 중 하나로 끝난다.

- 하린의 선택이 바뀐다.
- 누군가의 침묵이 단서가 된다.
- 현재 사건이 하린에게 다시 돌아온다.
- 하린이 분석관과 분석 대상 사이에서 흔들린다.
- 도윤과 하린의 방식 차이가 다음 사건의 동력이 된다.

단순한 코드 공개, 이름 공개, 숫자 공개로 끝내지 않는다.

## 11. 최종 기준

윤하린 구간의 핵심 질문은 이것이다.

> 윤하린은 진실을 분석하는 사람인가, 아니면 누군가가 남긴 진실의 일부인가?

모든 사건과 단서는 이 질문으로 돌아와야 한다.
