# 공개 금지 / 공개 가능 목록

## 목적

각 화에서 어디까지 말할 수 있는지 정리한다.
절대울지마는 진실을 빨리 말하는 이야기가 아니다.
선택의 흔적을 따라가며, 독자가 먼저 불안을 느끼게 해야 한다.

## 13~20화 공개 가능

- 윤하린이 사건 안에 있다는 감각.
- D-17이라는 구형 태그.
- 보호자 유도 발화라는 문구.
- 지정 배정이 하린에게 반복된다는 사실.
- 분석자 일치, 본인 반응값 대조.
- G.D.H.라는 외부 개입자 코드.
- `괜찮지 않으면`이라는 미완성 문장.

## 13~20화 공개 금지

- G.D.H.가 강도현이라고 확정.
- 강도현이 실제로 세션을 차단했다고 확정.
- 윤하린 기억 분리의 정확한 절차.
- RETURN 사건의 실제 구조.
- 한서연의 보류 선택 의미.
- 노아의 판단 불가.
- 방주 정체.
- 외계 문명 기록.
- 최종 질문.

## 21~30화 공개 가능

- 이재훈이 D-17과 미완성 문장을 알고 있음.
- 송유진도 일부 알고 있으나 전부는 아님.
- D-17이 감정 키가 아니라 과거 절차 용어였을 가능성.
- `분리 보류`라는 절차 문구.
- 도윤의 능력이 하린 사건에 필요하다는 점.
- 하린이 분석자가 아니라 반응 대상처럼 취급되고 있다는 점.
- 현재 사건과 하린 과거가 같은 절차 언어를 공유한다는 점.

## 21~30화 공개 금지

- 하린이 정확히 어떤 절차를 당했는지 확정.
- 강도현의 의도 확정.
- 이재훈의 과거 행동 전부.
- RETURN 생존자 침묵의 전체 이유.
- 강도현/임준서/한서연의 선택 충돌 전체 구조.
- 노아의 최종 결론.
- 방주가 무엇인지.
- 결말 철학.

## 문장 처리 원칙

공개 가능 정보도 설명하지 않는다.

나쁜 예:
- D-17은 과거 기억 분리 절차에서 쓰인 코드였다.

좋은 예:
- 종이 귀퉁이에 `D-17`이 보였다.
- 그 아래에는 `보호자 유도 발화`가 있었다.
- 접힌 줄 사이로 `분리 보—`까지만 보였다.
- 이재훈이 손으로 종이를 덮었다.

## 인물별 말할 수 있는 범위

### 윤하린

말할 수 있음:
- 업무상 확인한 사실.
- 모순.
- 기록 필요성.
- 절차상 반박.

말하면 안 됨:
- 자기 감정 직접 설명.
- “나는 두렵다 / 흔들린다” 같은 감정 라벨.
- 과거 기억을 확정하는 말.

### 임도윤

말할 수 있음:
- 선택의 흔적.
- 망설임.
- 남은 흔적.
- “이건 소리가 아니라 선택입니다” 같은 도윤식 표현.

말하면 안 됨:
- 하린의 과거를 단번에 읽어내기.
- 예언자처럼 진실을 맞히기.
- 감정적으로 과하게 위로하기.

### 이재훈

말할 수 있음:
- “지금은 말할 수 없습니다.”
- “그 이름은 아직 부르지 마십시오.”
- “혼자 보지 마십시오.”

말하면 안 됨:
- 친절한 과거 설명.
- RETURN 사건 강의.
- 강도현의 의도 확정.

### 송유진

말할 수 있음:
- 절차상 중단.
- 하린을 막는 현실적 이유.
- “네 반응값을 누가 쓰고 있어.”

말하면 안 됨:
- 모든 비밀을 알고 있는 듯 행동.
- 긴 설명.
- 과도한 감정 고백.

### 박현우

말할 수 있음:
- 구형 사전/소문/내부 자료 일부 발견.
- 불안한 농담.
- 독자 대신 묻는 질문.

말하면 안 됨:
- 핵심 진실 발견자 역할 과다.
- 분위기 깨는 농담 남발.

### 이나경

말할 수 있음:
- 시스템상 불가능한 경로.
- 태그/분류표의 모순.
- 논리적 반박.

말하면 안 됨:
- 감정 해설.
- 하린의 내면을 대변.
