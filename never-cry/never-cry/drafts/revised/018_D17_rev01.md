# 18화. D-17

- 상태: 수정본
- 수정 이유:
  - 기존 통화 사건 버전은 윤하린 에피소드로 기능하지 않음
  - 단서가 하린에게 돌아오지 않음
  - D-17을 현재 사건의 구체 단서로 재설계
- 회차 기능:
  - 평범한 재분류 사건에서 D-17 태그가 등장한다.
  - D-17이 구형 내부 사전상 `보호자 유도 발화`로 확인된다.
  - 현행 분류표에 없는 태그가 하린 앞에 나타난다.
- 공개 가능:
  - D-17
  - 보호자 유도 발화
  - 현행 분류표에 없음
  - 2281년 이후 사용 중지 암시
- 공개 금지:
  - G.D.H. 정체 확정
  - 강도현 직접 연결
  - 윤하린 기억 분리 확정
