# drafts 폴더 사용법

## 목적

이 폴더는 절대울지마 원고의 초안, 수정본, 폐기본을 보관한다.

확정 원고는 이 폴더에 두지 않는다.  
확정 원고는 `episodes/final/`로 이동한다.

---

## 폴더 구조

```text
drafts/
├─ raw/        # 최초 초안
├─ revised/    # 피드백 반영 수정본
└─ discarded/  # 폐기본
```

---

## 사용 기준

### raw

처음 쓴 원고를 저장한다.

```text
drafts/raw/022_같은사건다른눈_v01.md
```

### revised

사용자 피드백이나 문체 수정 후 저장한다.

```text
drafts/revised/022_같은사건다른눈_rev01.md
```

### discarded

버린 원고를 저장한다.

```text
drafts/discarded/022_같은사건다른눈_폐기_설명형.md
```

---

## 폐기본을 남기는 이유

폐기본은 실패작이 아니라 학습 자료다.

특히 아래 이유로 폐기된 원고는 반드시 보관한다.

- 웹툰 설명형
- 사건 기능 부족
- 단서가 하린에게 돌아오지 않음
- 설정 설명 과다
- 관계 감정 조기 과열
- 공개 금지 정보 노출

---

## 핵심 문장

> 초안은 버려도 된다.  
> 버린 이유는 버리면 안 된다.
