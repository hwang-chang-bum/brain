# RETURN 핵심 문서

## RETURN 12인의 선택 축

RETURN 사건은 전쟁이 아니다.  
선악 구도가 아니다.  
각자 다른 방식으로 인류를 살리려 한 선택의 충돌이다.

## 봉인파

대표: 강도현

주장:

- 인류는 준비되지 않았다
- 방주 정보는 봉인해야 한다
- 보호를 위해 통제가 필요하다

관련 인물:

- 강도현
- 박성우
- 이재훈
- 최유진

## 공개파

대표: 임준서

주장:

- 인류는 선택할 권리가 있다
- 진실은 숨기면 안 된다
- 위험해도 알아야 선택할 수 있다

관련 인물:

- 임준서
- 김태윤
- 유현준

## 보류파

대표: 한서연

주장:

- 아직 판단할 수 없다
- 방주를 이해하지 못했다
- 인간이 감당할 수 있는지 확인해야 한다

관련 인물:

- 한서연
- 오민석
- 서나경
- 정세아

## 노아

노아는 어느 편도 선택하지 못했다.  
최종 결론은 판단 불가.  
결정권은 인간에게 반환되었다.

## RETURN 사건의 핵심

- 강도현은 인류를 지키고 싶었다
- 임준서는 인류를 믿고 싶었다
- 한서연은 인류가 감당할 수 있는지 알고 싶었다
- 노아는 판단하지 못했다

## 공식 인식

공식 명칭: RETURN 연구소 사고
공식 결론: 대형 연구시설 사고
공식 원인: 실험 시스템 오류
공식 결과: 사망자 다수, 시설 파괴, 데이터 소실, 프로젝트 종료

하지만 실제는 다르다.

- 사망자 수 불명
- 실종자 다수
- 일부 생존
- 진실 봉인

## 현재 사회 인식

일반 시민에게 RETURN은 오래된 사고다.  
청년 세대 대부분은 잘 모른다.  
기억보존청 내부에서는 수상한 과거로 남아 있다.

## 집필 주의

- RETURN을 미스터리처럼 광고하지 않는다
- 잊힌 사건처럼 다룬다
- 고참들이 말하지 않는 분위기로 보여준다
- 진실은 자료가 아니라 사람들의 침묵에서 새어 나오게 한다
