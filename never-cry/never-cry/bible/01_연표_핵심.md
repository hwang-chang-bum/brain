## RETURN 사건

- 2281년 10월 17일: RETURN 최종 심의일
- 당시 윤하린 13세
- 당시 임도윤 12세

RETURN 사건은 다음이 아니다.

- 테러 아님
- 폭발 사고 아님
- 외계 침공 아님
- 단순 연구 사고 아님

실제 원인은 선택 충돌이다.

- 강도현: 봉인
- 임준서: 공개
- 한서연: 보류
- 노아: 판단 불가

## 2281년 사건 이후

- 윤하린 구간에서는 2281년과 연결되는 흔적만 남긴다
