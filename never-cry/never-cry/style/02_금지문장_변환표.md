# 금지문장 변환표

## 목적

자주 반복되는 웹툰 설명형 문장을 웹소설 문장으로 바꾸기 위한 변환표다.

집필 중 아래 금지문장이 보이면 즉시 수정한다.

---

## 행동 나열형

| 금지 문장 | 대체 방향 |
|---|---|
| 하린은 화면을 보았다. | 화면의 문장이 하린의 손을 멈추게 했다. |
| 하린은 멈췄다. | 다음 문장을 쓰려던 손끝이 빈칸 위에서 굳었다. |
| 송유진은 말했다. | 송유진의 말이 먼저 길을 막았다. |
| 박현우가 물었다. | 박현우의 농담기 없는 질문이 사무실을 더 조용하게 만들었다. |
| 이나경은 고개를 끄덕였다. | 이나경은 반박하지 않았다. 그것만으로도 충분한 동의였다. |
| 아무도 대답하지 않았다. | 모르는 침묵이 아니었다. 말해선 안 되는 쪽의 침묵이었다. |

## 감정 직접형

| 금지 문장 | 대체 방향 |
|---|---|
| 하린은 불안했다. | 하린은 같은 줄을 세 번 읽고도 저장하지 못했다. |
| 하린은 화가 났다. | 하린의 문장이 짧아졌다. |
| 하린은 혼란스러웠다. | 커서가 엉뚱한 줄에서 오래 깜빡였다. |
| 송유진은 걱정했다. | 송유진은 화면보다 먼저 하린의 손을 봤다. |
| 도윤은 신경 쓰였다. | 도윤은 묻지 않기로 한 질문을 끝내 지우지 못했다. |
| 이재훈은 죄책감을 느꼈다. | 이재훈은 오래된 봉투를 꺼내고도 열지 못했다. |

## 설정 설명형

| 금지 문장 | 대체 방향 |
|---|---|
| D-17은 보호자 유도 발화 태그였다. | 구형 사전의 D-17 옆에는 ‘보호자 유도 발화’라는 짧은 줄만 남아 있었다. |
| G.D.H.는 강도현의 이니셜이었다. | 세 글자는 아직 이름이 아니었다. 하지만 송유진은 이미 이름처럼 피하고 있었다. |
| 하린은 강도현의 양녀였다. | 하린은 ‘보호자’라는 단어 앞에서 늘 한 박자 늦었다. |
| 이재훈은 RETURN 생존자였다. | 이재훈은 2281년이라는 숫자를 설명하지 않고, 늘 먼저 치웠다. |
| 도윤은 선택의 흔적을 읽는다. | 도윤은 기록보다 먼저, 누군가 망설였던 자리를 봤다. |

## 관계 설명형

| 금지 문장 | 대체 방향 |
|---|---|
| 하린과 도윤은 서로를 신뢰하기 시작했다. | 하린은 도윤의 판단을 지우지 않고, 자기 기록 아래에 남겨두었다. |
| 송유진은 하린을 보호했다. | 송유진은 하린 앞의 화면을 닫고도 변명하지 않았다. |
| 이나경은 하린을 인정했다. | 이나경은 “오류가 아닙니다”라고 말한 뒤, 하린의 문장을 수정하지 않았다. |
| 박현우는 분위기를 풀었다. | 박현우가 웃기려 했지만, 아무도 웃지 않았다. 그는 그제야 웃음을 접었다. |
| 이재훈은 하린을 아꼈다. | 이재훈은 하린에게 답을 주지 않았다. 대신 위험한 질문을 먼저 알아챘다. |

## 엔딩 떡밥형

| 금지 문장 | 대체 방향 |
|---|---|
| 그리고 충격적인 단서가 드러났다. | 단서는 드러난 게 아니라, 지워지지 않고 남았다. |
| 하린은 큰 비밀에 다가가고 있었다. | 하린은 자신이 문 앞에 선 줄 알았지만, 문은 이미 안쪽에서 잠겨 있었다. |
| 이 사건은 하린과 관련이 있었다. | 사건 파일의 대상자는 타인이었다. 그런데 분석자 이름이 먼저 눈에 들어왔다. |
| 모든 것은 2281년과 연결되어 있었다. | 2281이라는 숫자는 아무 설명도 없이, 자꾸 같은 자리로 돌아왔다. |

## 수정 원칙

1. “했다” 문장이 3번 연속 나오면 리듬을 바꾼다.
2. 감정 단어가 나오면 행동·호흡·문장 선택으로 바꾼다.
3. 설정을 설명해야 할 때는 파일, 봉투, 화면, 단어 조각으로 보여준다.
4. 떡밥은 크게 외치지 말고 작게 남긴다.
5. 인물의 관계는 말로 정리하지 말고, 선택의 방식으로 드러낸다.
