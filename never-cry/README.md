# 절대울지마 / Never Cry Story Brain

이 폴더는 소설 **절대울지마**의 설정, 집필 규칙, 에피소드 진행 기준을 보존하기 위한 작업 저장소다.

## 사용 원칙

새 채팅방 또는 새 집필 세션을 시작할 때는 아래 순서로 확인한다.

1. `rules/01_작풍_금지사항.md`
2. `rules/02_웹소설_문체규칙.md`
3. `episodes/013_020_윤하린구간_정리.md`

## 현재 고정 상태

- 현재 시점: 2296년
- 12화 종료
- 13화부터 윤하린 중심 구간 시작
- 13~20화 목표: 윤하린이 사건 안에 있다는 사실을 독자가 눈치채는 구간
- 진실 확정 공개 금지
- 방주, 노아, 외계문명, 최종 철학 직접 공개 금지
- 새 인물 생성 금지
- 새 설정 추가 전 충돌 검증 필수
