# 설정 충돌 검증 규칙

## 새 설정 추가 전 확인

새로운 사건, 인물, 조직, 능력, 과거사를 넣기 전 반드시 확인한다.

1. 기존 연표와 충돌하는가?
2. 인물의 나이와 당시 시점이 맞는가?
3. 기존 관계사전과 감정선이 충돌하는가?
4. RETURN 사건의 선택 축을 흐리는가?
5. 후반 공개 예정인 진실을 너무 빨리 말하는가?
6. 새 인물이 기존 인물의 역할을 빼앗는가?
7. 설정이 사건보다 앞서고 있지 않은가?

## 새 인물 제한

핵심 인물은 새로 만들지 않는다.
단발 사건의 민원인, 보호자, 대상자는 가능하다.
다만 단발 인물이 장면을 오래 차지하면 안 된다.

## 새 조직 제한

기억보존청, 특수기억해석실, 기억복원국, RETURN 관련 조직 외의 새 조직은 신중히 추가한다.
필요할 경우 기존 조직의 하위 부서로 처리한다.

## 능력 설정 제한

능력은 인물의 선택을 드러내기 위한 장치다.
능력 배틀처럼 쓰면 안 된다.

- 임도윤: 선택의 흔적을 읽음
- 윤하린: 모순을 감지함

두 능력은 정답을 내는 능력이 아니다.
각자 부족한 방식으로 세계를 보는 능력이다.

## RETURN 관련 공개 제한

다음은 초중반 직접 공개 금지다.

- 방주의 정체
- 노아의 정체와 최종 역할
- 외계문명 기록의 구체 내용
- RETURN 사건의 전체 진실
- 강도현의 최종 선택 이유
- 한서연의 보류 이유 전체

가능한 것은 흔적뿐이다.

- 누락된 기록
- 말하지 않는 어른들
- 이상한 태그
- 지정 배정
- 회수 명령
- 보류라는 단어의 반복

## 윤하린 구간 충돌 주의

13~20화 이후에도 윤하린의 비밀을 너무 빨리 확정하면 안 된다.

가능:

- 하린이 사건 안에 있다는 암시
- D-17, 보호자 유도 발화, 분리 보류 같은 단서
- 이재훈과 송유진의 침묵
- 도윤과의 불편한 공동 검토

금지:

- 강도현이 직접 모든 것을 설명
- 이재훈이 친절하게 과거를 해설
- 하린이 한 번에 모든 기억을 복원
- RETURN 사건의 원인 강의
